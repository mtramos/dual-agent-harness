Dual Agent / how it works
Getting started
API keys
Paste your Claude key (console.anthropic.com) and Perplexity key (perplexity.ai/api-platform) into the key bar. They save automatically and persist next time you open this file. Keys stay local — never sent anywhere except directly to each API.
Session title
Name your session — it becomes the export filename.
Hypothesis
Write what you’re testing. Included in every export so sessions are self-documenting.
Input options
Type / paste
Type a topic, paste an argument, or paste research directly into the input box.
↑ Attach document
Upload a .txt, .md, or .pdf file. Its content is injected as context into both agent calls alongside your message.
Model selector
Choose the model per panel from the dropdown in each panel header. Claude options: Haiku 4.5, Sonnet 4, Opus 4. Perplexity options: Sonar, Sonar Pro, Sonar Reasoning Pro, Sonar Deep Research.
Lens chips + role prompt
Lens chips
Multi-select chips that append a specific angle to the role prompt. Critic finds weaknesses. Steel-man argues the strongest version. Devil’s advocate argues the opposite. Find the gaps looks for missing evidence.
Role prompt
Defines the role both agents play. Always visible and editable. Chips append to it — they don’t replace it. Write your own: “You are a philosopher. Challenge every assumption with first principles thinking.”
Flagging responses
Flags appear under each response, tinted in that panel’s color. Click to toggle. All flags are counted in the score bar and included in exports.
↑ strong
Strong counterargument — your essay needs to address this.
↓ weak
Weak or easily rebutted. Low priority.
★ key
Key insight worth keeping.
⇄ diverge
Marks where this response diverges from the other agent. Highlights in violet.
Diff + export
⇄ Diff
Fires a third Claude call comparing the last response from each agent and surfaces genuine contradictions. Appears as a violet banner in both panels.
↓ .md
Downloads a Markdown file with hypothesis, active chips, selected models, flag summary, system prompt, and full conversation. Designed to paste back into Claude as test evidence.
↓ .json
Full structured export including all histories, flags, scores, and metadata.
